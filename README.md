# 🌏 全球生物猜謎探險 (Global Bio-Guesser)

這是一個專為國小 3-4 年級設計的互動式生態教育遊戲，結合了 **Google Gemini AI** 的強大對話能力與 **Google Apps Script (GAS)** 的輕量化部署。玩家將化身探險家，透過與 AI 探險隊長的問答，解開世界各地的生物之謎。

## ✨ 遊戲特色功能

### 1. 結構化百科排版 🧬
當玩家完全猜對生物名稱後，系統會自動生成結構化的百科小卡，包含：
* **地理位置**：說明該生物在該地區的棲息地。
* **生物特徵**：列點呈現該生物的生理構造或特殊能力。
* **有趣小知識**：分享冷知識，提升學習趣味。

### 2. 智慧求救提示系統 💡
卡關了嗎？點擊左下角的 **「提示」** 按鈕，AI 隊長會提供關於飲食習慣或棲息環境的科學線索，且**提示不計入提問次數**，鼓勵學生思考而非放棄。

### 3. 多國探險地圖 🗺️
支援切換多個生態系，包括：
* 🇹🇼 臺灣 (Taiwan)
* 🇯🇵 日本 (Japan)
* 🦁 非洲大草原 (Africa)
* 🐨 澳洲 (Australia)
* 🐍 亞馬遜雨林 (Amazon)

### 4. 最佳紀錄儲存 🏆
系統會自動記錄每局的提問次數，並儲存在玩家的瀏覽器中，激勵學生用更精準的提問（例如：「你是脊椎動物嗎？」）來打破紀錄。

## 🛠️ 技術架構與部署

本專案由兩個核心檔案組成：
* **`index.html`**：前端介面，負責 UI 排版、對話流程控制及本地紀錄儲存。
* **`code.gs`**：後端處理，透過 Google Apps Script 連接 Gemini API 並設定系統指令（System Instruction）。

### 如何更新遊戲邏輯？
1. 開啟您的 **Google Apps Script** 專案。
2. 更新 `code.gs` 中的 `sysRole` 指令，即可調整 AI 的說話語氣或排版格式。
3. 重新執行 **「部署」 > 「管理部署」 > 「編輯」 > 「新版本」** 以使更改生效。

## 📝 課堂教學應用 (教師指南)

本遊戲特別適合配合 **AI 生態學習單**：
1. **觀察與紀錄**：引導學生將 AI 生成的「結構化排版」內容填入學習單的生物特徵欄位。
2. **提問訓練**：練習使用「二分法」提問（如：是/不是），學習分類學的基本概念。
3. **資訊過濾**：學生需在 AI 提供「提示」後，過濾出關鍵線索來推論答案。
