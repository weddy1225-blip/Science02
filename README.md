🌍 世界生態之旅：AI 導遊猜謎遊戲
這是一個專為 國小四年級自然科學課程 設計的互動式 AI 學習工具。透過與 AI 導遊的對話，學生可以練習觀察生物特徵，並理解生物如何透過生理構造適應全球不同的氣候環境。

✨ 遊戲特色
智能 AI 導遊：使用 Google Gemini 1.5 Flash 模型，扮演充滿活力的探險導遊。

全球生態探索：隨機從世界各地（如：南極、亞馬遜、撒哈拉）挑選具代表性的生物。

同步計分系統：即時統計「探索次數」，並具備「最佳紀錄」保存功能。

科學講評：當學生答對時，AI 會針對學生的提問策略進行分析，並講解該生物的氣候適應知識。

猜測排除邏輯：系統會自動判定「猜答案（如：你是企鵝嗎？）」與「提問特徵（如：你住的地方冷嗎？）」的差異，確保計次精準。

🛠️ 技術架構
前端：HTML5, CSS3 (Flexbox/Grid), JavaScript (Fetch API)。

後端：Google Apps Script (GAS)。

AI 模型：Google Gemini 1.5 Flash API。

部署平台：GitHub Pages。

🚀 快速開始
1. 後端設定 (GAS)
建立一個新的 Google Apps Script 專案。

貼入 Code.gs 中的程式碼。

在「指令碼屬性」中新增 GEMINI_API_KEY。

點擊「部署」>「網頁應用程式」，權限設定為 「所有人 (Anyone)」。

複製產生的 網頁應用程式 URL。

2. 前端設定 (GitHub)
在 GitHub 建立新儲存庫 (Repository)。

上傳 index.html。

將 index.html 中的 const GAS_URL = '...' 替換為你的 GAS 網址。

前往 Settings > Pages，開啟 GitHub Pages 功能。

🎮 玩法說明
開啟網頁：看到 AI 導遊的歡迎詞後即可開始。

提問特徵：輸入關於外觀、食性、棲地的問題（例如：「你住的地方會很熱嗎？」）。

觀察計次：每問一個特徵問題，上方的「探索次數」會增加 1。

大膽猜測：輸入「你是 [動物名稱]」，若答對，AI 會給予專屬講評。

挑戰紀錄：嘗試用最少的提問次數猜中答案，刷新你的「最佳紀錄」！
