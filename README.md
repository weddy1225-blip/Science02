## ✨ 遊戲特色

* **智能 AI 導遊**：使用 Google Gemini 1.5 Flash 模型，扮演熱情的探險導遊，帶領學生環遊世界。
* **全球生態探索**：隨機從世界各地（如：南極冰原、非洲草原、亞馬遜雨林）挑選代表性生物。
* **同步計分系統**：具備「探索次數」即時統計與「最佳紀錄」保存功能。
* **科學講評回饋**：答對時，AI 會針對學生的提問策略進行分析，並講解生物適應氣候的關鍵生理特徵。
* **精準計次邏輯**：系統自動排除「指令」與「猜答案」的對話，確保計數器與 AI 講評完全同步。

## 🛠️ 技術架構

* **前端介面**：HTML5 / CSS3 (Responsive Design) / JavaScript
* **後端邏輯**：Google Apps Script (GAS)
* **核心引擎**：Google Gemini 1.5 Flash API
* **佈署環境**：GitHub Pages

## 🚀 快速開始 (開發者佈署)

### 1. 後端設定 (GAS)
1.  建立一個新的 [Google Apps Script](https://script.google.com/) 專案。
2.  貼入本專案提供的 `Code.gs`。
3.  在「指令碼屬性」中新增 `GEMINI_API_KEY`。
4.  點擊「部署」>「網頁應用程式」，權限設定為 **「所有人 (Anyone)」**。
5.  複製產生的 **Web App URL**。
